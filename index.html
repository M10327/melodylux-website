About Computer
x

This is the third website design I've made, and the most complex one by far. I've had to learn a good amount of javascript html and css to put this together. I write everything from scratch in local files (first using notepad++ and now using vs code), so I keep everything as basic as I can. This is currently just a hobby for me, as my main coding preference is with c#.

This design was heavily inspired by Warframe 1999. The background music, login sfx, and button clicks are all taken from the game. I'm sure some icons look a little out of place, but I'm not exactly a graphic designer so I make due with what I can find online.

Hopefully it works as intended on all systems, but I'm not very good at making websites that properly scale with screens yet so I imagine it doesn't play well on mobile or weird resolutions.
